# FreeNote 重构计划：苹果极简主义风格

## 核心设计理念
1. **极致简约**：大量留白，去除多余的边框和背景色，聚焦内容。
2. **磨砂玻璃 (Glassmorphism)**：在顶部导航和悬浮组件上使用 `backdrop-blur` 效果。
3. **精致细节**：使用更细腻的阴影、圆角（通常为 10-12px）以及 Apple 风格的系统字体。
4. **流畅动效**：通过 CSS 过渡和微动效提升交互质感。

## 技术栈升级
- **CSS 框架**：引入 **Tailwind CSS**（苹果风格精准控制的利器）。
- **组件库**：由 Element Plus 切换为 **Shadcn Vue** (基于 Radix Vue)，提供更现代、可高度自定义的组件。
- **图标库**：使用 **Lucide Icons**，线条更细、更符合极简美学。

## 实施路线图

### 第一阶段：环境搭建与基础配置
1. 安装并配置 Tailwind CSS 及其插件。
2. 引入字体系统（优先使用系统默认 San Francisco 风格字体）。
3. 配置全局主题色（极简黑白灰及苹果特有的蓝色/深灰点缀）。

### 第二阶段：布局与导航重构
1. **顶部导航栏**：重写 App.vue，实现吸顶磨砂效果。
2. **页面脚架**：重新设计 EditorPage.vue 的布局，让编辑器居中并优化最大阅读宽度。

### 第三阶段：核心功能组件美化
1. **编辑器外壳**：美化 MarkdownEditor.vue，优化行高和字体清晰度。
2. **工具栏重构**：将现有的 AI 功能按钮（总结、润色等）重构为精致的悬浮式或吸附栏。
3. **状态反馈**：使用 Shadcn 的 Toast/Dialog 替换 Element Plus 的消息提示。

### 第四阶段：验证与优化
1. 确保现有的本地文件读写（FileSystem API）功能无损。
2. 优化响应式表现，确保在不同屏幕下都有苹果般的优雅感。
